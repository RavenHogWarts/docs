# Quick Path

::: tip Feature Introduction
1. Easily copy path(s) of single or multiple documents
2. Easily copy folder paths
3. Easily insert document or folder paths into documents
:::

## Copy Path
In Obsidian's file tree, right-click on a file or folder, or right-click on multiple selected files to copy their paths

## Insert Path
In document editing mode, right-click in the document editing area to insert document or folder paths

## Commands
This tool provides the following commands. Enter `quickPath` in the command palette to see these options:
1. Copy Document Path: Copies the path of the currently active document
2. Copy Folder Path: Copies the path of the currently active folder

## Settings
1. Add insert path options to the editor menu
2. Choose between absolute or relative paths. Absolute paths start from the operating system root directory, while relative paths start from the current Obsidian vault root directory
3. Choose the separator between multiple paths, supporting newline, comma, semicolon, or space
