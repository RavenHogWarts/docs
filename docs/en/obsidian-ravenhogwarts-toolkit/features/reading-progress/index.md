# Reading Progress

::: tip Feature Introduction
1. Displays a reading progress ring for documents
2. Displays document table of contents
3. Supports returning to cursor position in edit mode, and returning to top/bottom of document in reading mode
:::

## Document Table of Contents
In edit mode, the active state of the table of contents depends on the cursor position

In reading mode, the active state depends on the page scroll position

### Toolbar
1. Quick Setting: Toggle floating table of contents display
2. Quick Setting: Overall position of reading progress
3. Quick Setting: Offset position of reading progress relative to the side
4. Copy TOC: Copy the current document's table of contents
5. Drag Width: At the edge of the floating TOC, drag to adjust its width

## Settings Options
1. Overall position of reading progress (left or right side)
2. Offset position of reading progress relative to the side (in px)
3. Toggle floating table of contents display
4. Whether to keep floating TOC always expanded
5. Toggle floating TOC toolbar
6. Display width of floating TOC (in px)
7. Toggle reading progress ring display
8. Toggle estimated reading time display
