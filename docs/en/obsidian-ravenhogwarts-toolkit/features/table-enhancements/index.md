# Table Enhancements

::: tip Feature Introduction
Process data in markdown tables with the following features:

1. Support for simple table editing
2. Support for using formulas to calculate data in tables and output to document properties
:::

## Formulas
### Features

1. Support for calculating data in tables and outputting to document properties
2. Support for saving formulas so they can be rerun when table data is updated

For details, see [Built-in Formulas](./formula.md)
