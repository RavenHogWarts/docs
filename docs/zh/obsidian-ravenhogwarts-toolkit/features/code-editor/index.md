# 代码编辑器

::: tip 功能介绍
1. 支持在obsidian中直接编辑代码文件
2. 支持代码高亮、代码折叠
3. 支持vscode编辑器的常用快捷键
:::

## 在代码编辑器中打开
在obsidian的文件树中，右键文件，可以选择在代码编辑器中打开（所有文件都可以）

## 创建代码文件
在obsidian的文件树中，右键文件夹，可以在当前文件夹中创建代码文件

代码文件创建窗口包括了文件类型的选择，文件名的输入，文件创建路径的预览，创建后是否在新标签页中打开等选项

### 文件类型
主要分为`自定义文件`和`其他常见代码文件后缀`

- 对于自定义文件，需要文件名的输入需要包含文件后缀，如`test.js`等

## 编辑器快捷键
本工具支持vscode编辑器的常用快捷键，如：

- `ctrl+/`：注释/取消注释
- `ctrl+f`: 查找
- `ctrl+h`: 替换
- `ctrl+z`: 撤销
- `ctrl+y`: 重做
- `ctrl+c`: 复制
- `ctrl+v`: 粘贴
- `ctrl+x`: 剪切
- `ctrl+a`: 全选
- `ctrl+[`: 代码行整体向左缩进
- `ctrl+]`: 代码行整体向右缩进
- `ctrl+shift+[`：折叠代码
- `ctrl+shift+]`：展开代码

## 命令
本工具提供以下命令，在命令面板中输入`codeEditor`会出现以下选项：
1. 创建代码文件，如果当前有激活文档，文件会创建在当前文档的同级目录下，否则会创建在当前obsidian库根目录下

## 设置页选项
1. 注册的文件扩展，注册后的文件单击会直接在obsidian中打开
2. 设置编辑器的主题，一共四种主题(vs，vs-dark，hc-light，hc-dark)
3. 是否显示编辑器的行号
4. 是否显示编辑器的缩略图
5. 设置编辑器中字体大小
6. 设置编辑器中字体
7. 设置一个`tab`键的字符数
8. 设置编辑器的行高
9. 设置编辑器的字间距
